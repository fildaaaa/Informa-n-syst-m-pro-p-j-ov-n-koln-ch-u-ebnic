### Informační systém pro půjčování školních učebnic

Jednoduchý informační systém pro evidenci školních učebnic, které si žáci půjčují a na konci roku vracejí.

Systém je určen učitelům a žákům. Projekt slouží pro usnadnění a zpřehlednění systému. Nahradí zdlouhavé papírování ohledně půjčování a celkově zrychlí proces

Administrátor/knihovnice - zapisuje učebnice a knihy, spracovává výpujčky knih, označuje poškození
Učitel - Vidí výpůjčky své třídy, kontroluje stav učebnic, pomáha s hromadným vydáním/vrácením učebnic
Student - Zobrazuje seznam půjčených učebnic, zjišťuje termín vrácení, hlásí poškození

# Spravovaná data
Učebnice: název, předmět, ročník, stav, identifikátor.
Žáci: jméno, příjmení, třída.
Výpůjčky: kdo má kterou učebnici, datum půjčení a vrácení, stav při vrácení.
Poškození: typ poškození, kdo jej způsobil, datum nahlášení.
Uživatelé (role): student, učitel, administrátor.
